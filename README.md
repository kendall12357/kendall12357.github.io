## How to update your GitHub website

1. Open your GitHub repository.
2. Delete or replace the old:
   - `index.html`
   - `style.css`
3. Upload the new:
   - `index.html`
   - `style.css`
4. Commit changes.
5. Wait a few minutes.
6. Refresh your website.
